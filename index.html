Classroom Monitoring System

Efficiently monitor your classroom with real-time data and analysis.

Features

This system is designed to monitor classrooms, providing real-time data on various parameters such as temperature, occupancy, and noise levels. The system ensures optimal classroom conditions, helping both students and teachers focus on education.

Attendance Tracking: Real-time attendance monitoring using cameras and microphones (with permission).

Engagement Monitoring: Tracks student attention and participation.

Quiz Participation: Tracks performance in quizzes designed by teachers.

Automated Notifications: Email and SMS reminders before class and real-time SMS alerts for non-engaged students.

Feedback System: Students provide feedback directly to teachers.
